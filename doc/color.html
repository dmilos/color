Description

Color models or what this library make this library.

Note

All color models are combination of template class color::model and next types as its template parameter :
  • std::uint8_t, std::uint16_t, std::uint32_t, std::uint64_t,
  • std::int8_t, std::int16_t, std::int32_t, std::int64_t,
  • float, double, long double,
  • color::type::split233_t, color::type::split323_t, color::type::split422_t, color::type::split556_t, color::type::split655_t,
  • color::type::uint24_t, color::type::uint48_t.
